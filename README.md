# 17k_com
**MyMongoDB.py：**
利用pymongo库封装MongoDB，方便调用。

**get_acw_sc__v2.js：**
每一个ip的cookie都具有时效性，当失效后需要重新获得cookie。

**get_free_books.py：**
获得多部小说的url并存储到csv文件中。

**get_chapter.py：**
先获得小说各个章节的url，再利用anyscio和aiohttp库实现异步请求小说章节内容，然后将数据储存到MongoDB中。
